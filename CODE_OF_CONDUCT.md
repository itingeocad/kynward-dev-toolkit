# Code of Conduct

Kynward Dev Toolkit should be a respectful and constructive space for contributors.

## Expected behavior

- Be respectful and professional.
- Discuss ideas, code, and documentation rather than attacking people.
- Give useful feedback with context and examples.
- Respect maintainer decisions and repository boundaries.
- Avoid sharing private information, confidential materials, or third-party content without permission.
- Use English for comments, issue descriptions, pull request descriptions, and documentation.

## Unacceptable behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing another person's private information.
- Spam, trolling, or repeated off-topic arguments.
- Attempts to bypass repository rules or review requirements.
- Submitting copyrighted or proprietary material without permission.
- Adding private game content, secrets, or unreleased Kynward materials.

## Enforcement

The maintainer may edit, hide, or remove comments, close issues, reject pull requests, block users, or take other reasonable actions to protect the project.

This code of conduct is intentionally simple while the toolkit is in early public setup stage. It may be replaced with a more formal community policy later.
