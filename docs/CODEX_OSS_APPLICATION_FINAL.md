## Short project description

Kynward Dev Toolkit is a public open-source toolkit for Godot prototype development workflows. It focuses on documentation checks, validation helpers, smoke-test templates, public-safe synthetic fixtures, and developer tooling practices that can be shared without exposing private game content.

## Longer project description

Kynward is a private fantasy world, settlement, travel, and generation simulation game project. Kynward Dev Toolkit is the public open-source companion repository that extracts only the safe developer tooling surface from that work.

The toolkit currently provides dependency-free Python checks for Markdown structure, local Markdown links, public synthetic JSON fixtures, and unit-style tests for the Markdown checkers. It also includes a public/private boundary checklist, usage documentation, smoke-test templates, GitHub issue templates, and a GitHub Actions workflow for continuous validation.

The repository is intentionally scoped so that tooling, documentation workflows, validators, and synthetic examples can be open-source while private lore, final game code, brand assets, narrative material, screenshots, music, and unreleased commercial data remain outside the public license.
