# Public Scope

This document defines what belongs in Kynward Dev Toolkit and what must remain outside this public repository.

## Public toolkit scope

The repository may include:

- small developer tools;
- documentation checkers;
- validation helpers;
- smoke-test checklists;
- data inspection utilities;
- synthetic public fixtures;
- examples that do not reveal private game content;
- maintainer workflow documentation;
- Godot support scripts that are useful outside the private Kynward game repository.

## Private project scope

The following must remain outside this repository unless explicitly approved and separately licensed:

- final Kynward game code;
- private lore;
- narrative canon;
- worldbuilding documents;
- campaign content;
- unreleased mechanics design;
- commercial art direction;
- brand identity files;
- logos, icons, music, sound, fonts, screenshots, and mockups;
- business plans and monetization details;
- internal roadmap details;
- secrets and deployment data.

## Synthetic examples only

Public examples should use synthetic names and neutral data.

Do not copy real private Kynward entities, factions, places, characters, maps, regions, or story materials into this repository.

## Review rule

If a contribution depends on private repository context, it should be rewritten as a generic public tool or rejected from this repository.
